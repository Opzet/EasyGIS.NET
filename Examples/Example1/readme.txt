Example 1

This project demonstrates ading an EGIS.Controls.SFMap control to a Windows Form application.
The application allows the user to select and open a shapefile, which is diplayed in the SFMap control.

- To pan around the map, click and drag the left mouse button on the map.

- To zoom into the map double click the left mouse button or use the mouse wheel.

- To zoom out of the map double click the right mouse button or use the mouse wheel.
