Example 8

This project demonstrates using an ICustomRenderSettings class to perform custom rendering of a shapefile added to an EGIS.Controls.SFMap control to a Windows Form application.

- To pan around the map, click and drag the left mouse button on the map.

- To zoom into the map double click the left mouse button or use the mouse wheel.

- To zoom out of the map double click the right mouse button or use the mouse wheel.

The example shapefile has been sourced from http://downloads.cloudmade.com.
the shapefile is based on data from OpenStreetMap and islicensed under the terms of the Creative Commons Attribution Share-Alike 2.0 license.
